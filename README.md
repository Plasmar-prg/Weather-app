# Погода в вашем городе

Приложение для просмотра текущей погоды и прогноза на неделю в любом городе мира.

## Стек технологий
- HTML5  
- CSS3 (адаптивный дизайн, анимации)  
- JavaScript (Vanilla JS)  
- OpenWeatherMap API  

## Как запустить локально
1. Скопируйте репозиторий или скачайте файлы  
2. Откройте `index.html` в браузере  
3. Введите название города и нажмите «Получить погоду»  

## Примечания
- Ключ API хранится в `config.js` и исключён из публичного доступа через `.gitignore`  
- Для разработки и тестирования используйте собственный ключ — замените в `config.js`  
- При первом открытии подставляется последний введённый город## Деплой
